Puerh，百姓网 UI 库。旨在建立稳定、简洁、兼容的前端框架。已经通过 Mac/Windows 下的 IE6+ / Firefox / Google Chrome 浏览器测试。Puerh 的目标是构建一套适合百姓网的前端解决方案。整套方案整合 TYPO.CSS 排版，基于与视觉稿一致的 980px Grid System 布局，包含各种 UI 组件，使用 Twitter Bootstrap 的 JavaScript 组件。

一、CSS 组件

Search(搜索框) PageNav（分页） Breadcrumb（面包屑导航） Headline（标题）
Group List（首页大列表） Catbox（图片列表） Tab Topbar（顶部导航）
Filter（类目过滤） Icons（小图标） Tag（小标签） Button（按钮）
Modal（弹出框） Dropdown（下拉菜单） Tips（可操作浮层） Alert Message（可操作消息）
Table（表格） Listing（列表） Images(图片展栏) Flow（流程）
Logo Navigation（导航） - -

二、JavaScript 组件

Modals(弹出层) Dropdowns（下拉菜单） Tabs（标签切换） Tips（浮层提示条）
Alert（可关闭消息） Typeahead（自动完成） Scrollspy（滚动生成） Lightbox（图片浏览浮层）
Validator（表单验证） Linkage（联动操作）
